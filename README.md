# Telegram-бот для конвертации валют

## Общее описание

Бот для конвертации валют через Telegram. Позволяет получать актуальные курсы и пересчитывать суммы из одной валюты в другую.

## Структура проекта

### 1. config.py

**Файл конфигурации:**

* Загружает переменные окружения из .env файла
* Содержит token - токен Telegram бота
* Словарь currencies с доступными валютами (рубль, доллар, евро, тенге, йена, юань)

### 2. extensions.py

**Модуль для работы с API валют:**

* APIException - пользовательское исключение для ошибок
* CryptoConverter.get_price(base, quote, amount) - основной метод конвертации:
* Проверяет корректность введенных валют и суммы
* Получает курсы с API exchangeratesapi.io
* Возвращает конвертированную сумму

### 3. main.py

Основной файл с логикой бота - bнициализация бота и диспетчера

## Команды

* /start, /help - инструкция по использованию
* /values - список доступных валют

Обработка сообщений с конвертацией (формат: <валюта1> <валюта2> <количество>)
